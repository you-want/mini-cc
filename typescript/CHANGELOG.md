# 更新日志

本文档记录 mini-cc 项目的所有重要变更。

## [未发布]

### 新增

- **GlobTool**: 文件模式匹配搜索工具
  - 支持标准 glob 语法（`*`, `**`, `?`, `[abc]`）
  - 自动忽略 `node_modules`、`.git` 等常见目录
  - 结果限制为 100 个文件，防止过载
  - 详细的教学注释和使用示例

- **GrepTool**: 内容搜索工具（支持正则表达式）
  - 支持正则表达式模式匹配
  - 可选的大小写敏感搜索
  - 支持文件类型过滤
  - 可显示匹配行的上下文（最多 3 行）
  - 返回匹配的行号和内容
  - 结果限制为 200 个匹配

- **FileEditTool**: 智能文件编辑工具
  - 基于精确字符串替换的编辑机制
  - 支持单次替换和全部替换模式
  - 自动验证匹配唯一性，防止误操作
  - 提供编辑前后的预览
  - 详细的错误提示和安全检查

- **WebFetchTool**: 网络请求工具
  - 支持 HTTP/HTTPS 协议
  - 支持 GET 和 POST 请求
  - 自定义请求头支持
  - 响应大小限制（50KB）
  - 30 秒超时保护
  - 自动处理错误和异常

### 改进

- 更新工具注册机制，添加工具分类注释
- 完善 `tools/index.ts` 的导出结构
- 所有新工具都包含详细的教学注释
- 统一的错误处理和日志输出

### 文档

- 新增 `docs/tools-guide.md` - 完整的工具使用指南
  - 所有工具的详细说明
  - 参数说明和示例

  - 工具组合使用示例

## 版本说明

本项目遵循[语义化版本](https://semver.org/lang/zh-CN/)规范。

### 版本格式

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 变更类型

- **新增**：新功能
- **改进**：对现有功能的改进
- **修复**：Bug 修复
- **废弃**：即将移除的功能
- **移除**：已移除的功能
- **安全**：安全相关的修复
- **文档**：文档更新
