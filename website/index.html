🌟 核心特性

麻雀虽小，五脏俱全。本项目不仅是一个玩具，更是一个功能完备的 Agent 原型。

🌐

多模型提供商支持

支持 Anthropic API 和 OpenAI 兼容接口，可自由切换 Claude、DeepSeek、Qwen、Kimi 等多种大模型。只需修改环境变量即可无缝对接。

⚡

流式输出与思考过程

原生支持 SSE (Server-Sent Events) 流式文本输出，并且完美适配了如 Qwen/DeepSeek 等模型的推理思考过程 (reasoning_content) 实时展示。

🛠️

强大的 Tool Use 系统

内置三大核心工具：
BashTool 执行系统终端命令。
FileReadTool 提取本地文件上下文。
FileWriteTool 覆盖写入文件，实现代码自动修改。

🔄

自主 Agent 事件循环

实现了经典的 ReAct 消息事件循环，模型可根据环境反馈多次连续调用工具，直到最终任务完成。

🚀 快速开始

以 TypeScript 版本为例，仅需几步即可在本地跑起你的专属 Agent。

1. 克隆与安装依赖

git clone https://github.com/you-want/mini-cc.git cd mini-cc/typescript pnpm install

2. 配置环境变量 (.env)

# 选项: 'anthropic' 或 'openai' PROVIDER=openai # 以 Qwen 为例的兼容接口 OPENAI_API_KEY=your_api_key_here OPENAI_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1 MODEL_NAME=qwen3.6-plus

3. 启动并交互

pnpm run dev

然后在终端输入你的需求，例如："帮我创建一个 hello.js 文件，内容是输出 Hello World。"
